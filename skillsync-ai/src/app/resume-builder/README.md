# AI Resume Builder

Part of **SkillSync AI**. React + Vite app for building resumes with AI-powered content (OpenRouter / Claude).

## Run from SkillSync AI root

```bash
# From repo root (SkillSync Ai/)
npm run dev:resume      # dev server
npm run build:resume    # production build
npm run preview:resume  # preview build
```

## Run from skillsync-ai folder

```bash
# From skillsync-ai/
npm run dev:resume
npm run build:resume
npm run preview:resume
```

## Run from this folder

```bash
cd skillsync-ai/resume-builder
npm run dev
npm run build
```

Set `VITE_OPENROUTER_API_KEY` in `.env` for AI features.
